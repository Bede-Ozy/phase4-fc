Building the Dashboard HTML

Now that our core configuration is done, let's build out the HTML structure for our public Dashboard page (index.html). We will rely on Tailwind CSS for a fully responsive, modern layout.

1. The Navigation Bar

First, replace the "Hello Phase 4 FC!" div from Chapter 1 with our sticky navigation bar. This bar will stick to the top of the browser as you scroll and feature a blurred glass effect.

        <div class="flex justify-between h-16 items-center">

Why leave elements empty?

You might notice elements like <div id="match-list"></div> are completely empty. This is because phase4fc is a dynamic application. Writing HTML inside these containers manually would take forever to update. Instead, we use Javascript (in Chapter 4) to fetch data from our database, generate the HTML, and inject it inside these specific ID containers.

3. Finalizing the file

Before closing the `body` tag, we'll need to link our Javascript file. We use type="module" because our JS architecture splits code across multiple files for maintainability. 

    <!-- Link at the bottom of the body tag -->
    <script type="module" src="js/app.js"></script>
</body>
</html>
